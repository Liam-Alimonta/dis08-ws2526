## 1) The Shell
**Quiz:** What is the exact output to the display when you type `echo Hello World`?  
**Answer:** Hello World

Quiz: Which command is used to find the directory you are currently in?
Answer: pwd

Quiz: If you are in /home/pete/Pictures and want to navigate to the parent directory (/home/pete), what is the full command you should use?
Answer: cd ..

Quiz: What command would you use to see hidden files?
Answer: ls -a

Quiz: How do you create a file called myfile?
Answer: touch myfile

Quiz: What command can you use to find the file type of a file?
Answer: file

Quiz: What command is used to display the contents of a file on the command line?
Answer: cat

Quiz: How do you quit out of the less command?
Answer: q

Quiz: What is the command to clear the terminal?
Answer: clear

Quiz: What flag do you need to specify to copy over a directory?
Answer: -r

Quiz: Using the mv command, how would you rename a file named cat to dog?
Answer: mv cat dog

Quiz: What command is used to make a directory?
Answer: mkdir

Quiz: How do you remove a file named myfile?
Answer: rm myfile

Quiz: What option should you specify for the find command to search by name?
Answer: -name

Quiz: How do you get quick command-line help for built-in Bash commands?
Answer: help

Quiz: How do you see the manual for a command?
Answer: man

Quiz: What command can you use to see a small description of a command?
Answer: whatis

Quiz: What command is used to create an alias?
Answer: alias

Quiz: What is the most common command to exit from the Linux shell?
Answer: exit
